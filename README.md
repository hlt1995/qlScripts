## 🗒️ 青龙面板升级以及依赖安装

恢复包解压完成切换容器后，输入`startalpine`进入Alpine

执行
```
cd /ql
ql update

```

面板升级完成后，执行安装NodeJS依赖
```
cd /ql/data/scripts
pnpm add axios sharp jsdom ds requests moment
```

执行安装Python3依赖
```
pip3 install request
```
